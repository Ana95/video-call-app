How to create account?

If you don't have account, you must first create one. To do this, click on button NEW USER?SIGN UP, that you can see on image below.



To finish process of registration, you need to entire information about your name, surname, username, email, password and phone number.



All of this field, which you can see on picture above, are required! If you don't entire data, you will get error message like on picture below.



If your username contains more than 15 charachters, you will get error message like on picture below.



If you entire incorrect email address, you will get notification about mistake you made!



Password is information that contains numbers, lower, upper case and special symbols, so if your password without some of these charachters, you will get message that entered value is too weak.



When you entire all necessary information in correct way, you will be successfully registered! Then you will receive message like on picture below!





How to edit your profile?

If you want to change some of information on your account, please click on the image the arrow points to!



How you can see on picture below, you can modify information about username, password, email and phone number and you can change your profile picture.



When you click on button UPDATE, dialog with data, you potentialy want to change, will appear.




How to make a call?

By click on menu item Calls like on image bellow, you can see a list of your friends.



If you want to start conversation with some of them, you only need to press button with handset.



When the button is pressed, you will see red button on screen, which you can activated anytime. Activated button means that the call ended.



You can start conversation with your friends, only when your friend accept your call, like on picture bellow.
